# 🎨 abd_app

This is a simple Flutter app I built while learning Flutter.
The idea is just to pick a color and apply it to the screen.

I also used this project to practice navigation between screens and basic UI design.

---

## 🚀 What the app does

* Enter your name and phone number
* Navigate between different screens
* Pick a color and preview it instantly
* Apply the selected color to a full screen
* View a list of favorite colors

---

## 🛠️ Built with

* Flutter
* Dart
* Material Design

---

## ▶️ How to run


git clone https://github.com/abdurrahman6565/abd_app.git
cd abd_app
flutter pub get
flutter run
